# Internal Team Task Tracker

**Technologies:** React, JavaScript, HTML, CSS

### Description
A lightweight tool for managing team tasks and tracking progress. This project extends a React starter template to include task status tracking and filtering for enhanced usability.

### Features
- Add, edit, and remove tasks
- Task status: Pending, In Progress, Done
- Filter tasks by status
- Responsive UI for desktop and mobile

### Installation
```bash
git clone <your-repo-url>
cd internal-task-tracker
npm install
npm start
```

### Usage
- Open the app in the browser at `http://localhost:3000`
- Add tasks using the input field
- Change task status using the dropdown
- Filter tasks using status buttons

### Notes
- Built by customizing an open-source React starter template.
- Focused on clean component structure and maintainable code.
